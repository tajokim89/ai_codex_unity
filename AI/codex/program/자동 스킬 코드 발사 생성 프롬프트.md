이 프로젝트는 유니티 2D 탑뷰 픽셀 아트 게임입니다.

요청 사항
Projectile.prefab(Sprite 폴더에 직접 생성 예정)에 붙일 ProjectileController 스크립트와 Player(GameScene에 이미 존재) 오브젝트에 붙일 AutoShooter 스크립트를 Scripts 폴더에 생성합니다.

작동 규칙
발사는 Player 오브젝트의 위치에서 시작됩니다.
발사체는 생성 시 설정된 이동 방향으로 Speed만큼 등속 이동을 합니다.
Lifetime 초가 지나면 자동으로 소멸됩니다.
2D 충돌을 사용해 Enemy 레이어와 접촉하면 대미지를 줘야 합니다.
플레이어의 입력 방향과 발사체의 발사 방향이 일치해야 합니다. 
정지 중에도 마지막으로 입력한 방향으로 계속 발사가 진행되어야 합니다.
발사체의 스프라이트는 기본적으로 오른쪽을 바라보고 있습니다.
왼쪽으로 발사 시 스프라이트가 플립되어야 합니다. 위, 아래, 대각선은 스프라이트의 회전으로 처리합니다.

예외 처리
적을 자동으로 추적하는 유도 기능 또는 관통 기능은 아직 만들지 않습니다.
방향을 전환할 때마다 과도하게 발사체가 생성되지 않도록 minTurnCooldown이 지나야 다음 발사가 진 행되어야 합니다.

요청 사항
Damage, Speed, Lifetime, Scale(발사체의 크기)은 Inspector 뷰에서 조정할 수 있어야 합니다.
Rigidbody 2D(물리 엔진)와 Colider 2D(충돌 판정) 사용을 가정합니다. 중력은 0으로 설정할 예정입 니다.
FireInterval(자동 발사 주기) minTurnCooldown(방향 전환 후 추가 지연, projectlePrefab(프리 참조), spawnPoint, baseRotationReference 등을 Inspector 뷰에서 조절 가능해야 합니다.
